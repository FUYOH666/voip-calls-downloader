# Changelog

Все значимые изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-01-16

### Added
- Поддержка автоматической загрузки записей звонков из CloudPBX Ростелеком
- Поддержка автоматической загрузки записей звонков из Связьтранзит
- Многопользовательский режим для CloudPBX (до 16 аккаунтов)
- Автоматическое отслеживание уже загруженных файлов (без дублирования)
- Фильтрация звонков по длительности и направлению
- Автоматический мониторинг и загрузка новых записей
- Настраиваемые интервалы проверки
- Health check и мониторинг состояния системы
- Поддержка автоматического перезапуска при сбоях
- Логирование всех операций
- Подробная документация для каждого проекта

### Changed
- Реструктуризация репозитория в монорепо с двумя проектами
- Улучшена структура документации согласно GitHub best practices

### Security
- Секреты и учетные данные хранятся только в .env файлах
- Все .env файлы исключены из git через .gitignore
- Примеры конфигурации без реальных данных
